Choosing fields with cut

  • use cut to "cut out" fields from data files
  • default is field based (with TAB field separator), with -f
    • use -d <char> to change the delimiter char
  • or, choose fields based on char ranges with -c
  • or, choose fields based on byte ranges with -b
